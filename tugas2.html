Langkah yang pertama adalah dengan mencari Equal Width Intervals yaitu membagi rentang dengan menggunakan nilai min dan max.

\(w=\frac{Zmax-Zmin}{k}\)

dimana

k = jumlah pengelompokkan

Sepal_Length

# mengubah diskritisasi sepal_length
def iris(min, max):
  k = 4
  w = (max - min)/k
  return (w)

iris(4.3, 7.9)

0.9000000000000001

Untuk melakukan diskritisasi maka diperlukan rentang. Untuk menghitung rentang dapat dicari dengan menggunakan

\(v = nilai min +w.i\)

dimana nilai min merupakan batas minimal, w adalah rentang dan i merupakan increment

#rumus rentang
def rentang_sepal_length(i):
  min = 4.3
  v = min + i * iris(4.3,7.9)
  return (v)

for col in ["sepal_length"]: 
  nilai = ds[col]
  def equal(nilai):
    if nilai >= 4.3 and nilai < 5.2:
      return "A"
    elif nilai >= 5.2 and nilai < 6.1:
      return "B"
    elif nilai >= 6.1 and nilai < 7.0:
      return "C"
    elif nilai >= 7.0 and nilai < 7.9:
      return "D"

ds["sepal_length"] = ds["sepal_length"].apply(equal)
