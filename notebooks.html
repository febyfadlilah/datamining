Menghitung Biner Simetris

Untuk menghitung jarak biner simetris maka diperlukan rumus

\[Rumus: d(i,j) = \ \frac{r+s} {q+r+s+t} \]
\[Keterangan: \]

  • r: i=1, j=0

  • s: i=0, j=1

  • q: i=1, j=1

  • t: i=0, j=0

def binersimetris(i,j):
  q=r=s=t=0
  for col in ['gender','test_preparation_course']:
    if data[col][i]==1 and data[col][j] == 1:
      q+=1
    elif data[col][i]==1 and data[col][j]==0:
      r+=1
    elif data[col][i]==0 and data[col][j] == 1:
      s+=1
    else :
      t+=1
  return (r+s)/(q+r+s+t)

Menghitung Data Nominal#

Atribut nominal mempunyai k kategori yang tidak memiliki urutan yang berarti. Jarak antara dua objek dengan m variabel adalah yang dibandingkan. Dengan p adalah banyaknya variabel yang mempunyai atribut yang sama.

\(d(i,j) = \frac{m-p}{p}\)

Dimana

  • m = jumlah data yang dibandingkan

  • P = jumlah data sama

# Menghitung jarak data Nominal
def d1(i,j):
    P = 4
    M = 0
    for col in ["race","parental_level"]:
        if data[col][i] == data[col][j]:
            M += 1

    return (P-M)/P

Dalam perhitungan kali ini menggunakan rumus Euclidian Distance untuk menemukan jarak pada type data numeric dengan menggunakan

\(d(i,j)=\sqrt{(x1-xj1)^2 + (xi2 - xj2)^2+.....}\)

# Menghitung jarak Numeric dengan metode Euclidean
def d2(x,y,h):
    result = 0
    for col in [hasil1, hasil2, hasil3]:
        result = math.pow(abs(col[x] - col[y]), h)
    return result**(1/h)

Langkah yang pertama adalah dengan mencari Equal Width Intervals yaitu membagi rentang dengan menggunakan nilai min dan max.

\(w=\frac{Zmax-Zmin}{k}\)

dimana

k = jumlah pengelompokkan

Sepal_Length

# mengubah diskritisasi sepal_length
def iris(min, max):
  k = 4
  w = (max - min)/k
  return (w)

Untuk melakukan diskritisasi maka diperlukan rentang. Untuk menghitung rentang dapat dicari dengan menggunakan

\(v = nilai min +w.i\)

dimana nilai min merupakan batas minimal, w adalah rentang dan i merupakan increment

Naive Bayes Classifier#

Naive Bayes adalah metode yang cocok untuk klasifikasi biner dan multiclass. Metode yang juga dikenal sebagai Naive Bayes Classifier ini menerapkan teknik supervised klasifikasi objek di masa depan dengan menetapkan label kelas ke instance/catatan menggunakan probabilitas bersyarat.

Naive Bayes

\(Rumus:\) $\(P(xⅈ|𝑦ⅈ)=\frac{1}{\sqrt2πδ^2}^e-\frac{(xi.μj)^2}{2δ^2j}\)$

\[δ=\sqrt\frac{Σ(xi-μ)^2}{N}\]
\[μ=\frac{Σᴺᵢ=1}{N}\]
\[P(x|c)P(c)\]

\(Example:\)

  1. P(x | Setosa) = cari semua standar deviasinya dan kalikan semua standar deviasinya

  2. P(Setosa) = Jumlah setosa dibagi jumlah data (\(\frac{50}{100}\))

  3. P= P(x | setosa) * P(Setosa)

K-Means Clustering#

K-means merupakan salah satu algoritma yang bersifat unsupervised learning. K-Means memiliki fungsi untuk mengelompokkan data kedalam data cluster. Algoritma ini dapat menerima data tanpa ada label kategori.

K means clustering merupakan metode algoritma dasar,yang diterapkan sebagai berikut

  • Menentukan jumlah cluster

  • Secara acak mendistribusikan data cluster

  • Menghitung rata rata dari data yang ada di cluster.

  • Menggunakan langkah baris 3 kembali sesuai nilai treshold

  • Menghitung jarak antara data dan nilai centroid(K means clustering) *Distance space dapat diimplementasikan untuk menghitung jarak data dan centroid. Contoh penghitungan jarak yang sering digunakan adalah manhattan/city blok distance
